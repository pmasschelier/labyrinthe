# Installation

```
git clone git@gitlab.enst.fr:2021INF103/groupe2/masschelier-paul-marie.git
cd masschelier-paul-marie
```
Pour lancer le programme
```
make run
```
Pour créer la documentation (entrée : doc/index.html) :
```
make doc
```
